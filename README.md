# Colony Portal  
A new frontend for colony that has GitHub integration.  

## Setup  
```  
$ npm install  
```  

## Running  
```  
$ npm start  
```  

## Running W/ Test Colony  
```  
$ npm run create:testColony  
$ npm start  
```  

## Build  
```  
$ npm run build  
```  

## Prettier  
```  
$ npm run prettier  
```  

## Generate test colony  
```  
$ npm run create:testColony  
```
